------------
Description
------------

Ce module customisé a pour but de donner une page d'informations récapitulatives pour les distributeurs.
Le module repose sur des configurations customisées, complexes, et assez avancées des modules Panels et Views.


------------
Installation
------------

Installer et activer le module comme n'importe quel module ordinaire.

Ce module va créer plusieurs Views pour permettre de récapituler les informations concernant les
distributeurs.
Il va également créer automatiquement une page-Panel ('espace_professionnel'), mettant en forme les
différentes vues relatés aux sdistributeurs.


Un guide pour les utilisateurs et un autre pour les administrateurs est disponible sur demande (contact email
mentionné plus haut).
